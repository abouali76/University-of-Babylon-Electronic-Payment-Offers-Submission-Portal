تم تجهيز هذا المجلد (COMP) ليكون نسخة كاملة ونظيفة من النظام، جاهزة للرفع على أي دومين آخر أو للعمل محلياً.

محتويات المجلد:
1. client: الكود المصدري للواجهة الأمامية (React + Vite).
2. server: الكود المصدري للخلفية (Node.js) مع قاعدة بيانات SQLite للعمل محلياً.
3. supabase: إعدادات ومخططات Supabase.
4. ملفات الصور الأساسية (الشعار والخلفيات).
5. ملفات SQL لتهيئة قاعدة البيانات.
6. Dockerfile و docker-compose للتشغيل باستخدام الحاويات.

طريقة التشغيل محلياً:

أولاً: الواجهة الأمامية (client)
1. ادخل إلى مجلد client.
2. قم بتشغيل الأمر: npm install
3. قم بتشغيل الأمر: npm run dev

ثانياً: الخلفية (server)
1. ادخل إلى مجلد server.
2. قم بتشغيل الأمر: npm install
3. قم بتشغيل الأمر: node index.js

ملاحظات هامة:
- تم استثناء المجلدات الكبيرة وغير الضرورية مثل (node_modules, dist, .git).
- ملفات الـ .env موجودة مسبقاً وتحتوي على الإعدادات الحالية للعمل مباشرة.
- نسخة قاعدة البيانات المحلية (database.sqlite) موجودة داخل مجلد server لضمان استمرارية البيانات.
- يمكنك تعديل ملفات .env عند الرفع على الدومين الجديد لتغيير عناوين الـ API أو قواعد البيانات.
